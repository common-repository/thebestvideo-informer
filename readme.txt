Custom select, thematic video-informer News Digest from TheBestVideo.net project.

== Description ==

This is video informer News digest from TheBestVideo.net project.
Find a lot of categories and tags. Over 800 channels now and even more soon. By default, widget displays last 3 images from the general list but you can choose any numbers.
Install this widget on your site and your visitors will always be informed about all the most interesting video news.
The widget correctly matches the design of your sites.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/thebestvideo` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add the widget to your sidebar from Appearance->Widgets
4. Set the specific category or tag to display images. Or use the default settings.
